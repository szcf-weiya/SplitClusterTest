# Calculate the accuracy (FDR, Power, F1 score) if truth is available

Calculate the accuracy (FDR, Power, F1 score) if truth is available

## Usage

``` r
calc_acc(est, truth)
```

## Arguments

- est:

  the selected relavent features

- truth:

  A vector of the true relevant features

## Value

a vector of size three: fdr, power, and F1 score
